# 📚 Articles Page

A simple and responsive Articles Page built using HTML, CSS, and Bootstrap.

## 🚀 Features
- Clean UI with category cards
- Responsive layout using Flexbox & Bootstrap
- Interactive sections with carousel
- Navigation between sections

## 🛠️ Technologies Used
- HTML5
- CSS3
- Bootstrap 4

## 📂 Project Structure
- index.html
- style.css

## 💡 What I Learned
- Structuring web pages using HTML
- Styling using CSS and Flexbox
- Using Bootstrap components like Carousel
- Debugging layout issues in browser
